\# Project Context



\## Architecture

\- Clean Architecture

\- Package by feature

\- No business logic in controllers

\- Domain isolated from frameworks



\## Stack

\- Java 21

\- Spring Boot

\- JPA

\- Testcontainers

\- Maven



\## Constraints

\- High availability

\- Auditability required

\- Idempotent APIs

\- JWT security



\## Coding Standards

\- SOLID required

\- No cyclic dependencies

\- Tests mandatory
